# Job Seeker
This is a web application for searching jobs. You can make your profile, search and apply for jobs.
# Tecnologies
- Back-end: Typescript, Node.js, Express and MongoDB for database.
- Front-end: Typescript and React.
# Features
## Users
- You can create your account and search and apply for jobs.
- You can see company details and job details
- You can delete your candidature if you need.
- You have profile page, there you can see info about you and every jobs that you are applied for.
## Admins
- Admin can register company and edit or delete this company.
- Admin can create and add jobs to the company and edit and delete these jobs.
- Admin can see all candidates for current job and accept or reject their candidatures.
- Admin also have profile page that he can see information about him and companies that he registrate.
# How to start the application
- Server: Open folder server and write command `npm run dev` in terminal to start the server.
- Client: Open folder client and again in terminal write comman `npm run dev` and you must see link that you can open.
- Data: In this folder you have all test data that you can add to your local database in MongoDB compass.
